# SSB (SKY SOUND BOX)
개발 중인 사이트의 이름입니다.


## 프로젝트 소개
자신이 직접 작곡한 음원 파일을 올려 사람들과 공유하고
다른 사용자들의 음원을 스트리밍 할 수 있는 음악 사이트입니다.


## 개발 환경 및 기술 스택

### 프로그래밍 언어 및 개발 환경:
* Java 버전: Java 17
* IDE: IntelliJ IDEA

### 백엔드 개발:
* 프레임워크: Spring Boot 3.0.6
* ORM (Object-Relational Mapping):
* * JPA (Java Persistence API)
* * JDBC Template
* 데이터베이스: MySQL

### 프론트엔드 개발:
* 프론트엔드 프레임워크: React
